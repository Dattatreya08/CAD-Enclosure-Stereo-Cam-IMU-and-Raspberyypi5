3D Printable Enclosure for Raspberry Pi 5, IMX219-83 Stereo Camera & WT901BLECL IMU

Overview

This repository contains the CAD files and documentation for a compact, 3D-printable enclosure designed for:

* Raspberry Pi 5
* Waveshare IMX219-83 Stereo Camera
* WT901BLECL Bluetooth 5.0 9-Axis IMU

The enclosure is intended for robotics, computer vision, Visual Odometry (VO), Visual-Inertial Odometry (VIO), and SLAM applications.

---

Features

* Compact rectangular enclosure
* Dedicated mounting for Raspberry Pi 5, Stereo Camera, and IMU
* Internal cable routing
* Ventilation openings for Raspberry Pi cooling
* Heat-set insert mounting for easy assembly
* Optimized for FDM 3D printing
* STEP and STL files included

---

Design Principles

The enclosure was designed based on the following principles:

* Proper spacing between components
* Consideration of EMI (Electromagnetic Interference)
* Efficient cable management
* Structural stability using 3 mm wall thickness
* Material optimization
* Easy assembly and maintenance
* Adequate ventilation for heat dissipation
* Compact and practical layout

---

Printing Recommendations

* Material: PLA or PETG
* Layer Height: 0.20 mm
* Nozzle: 0.40 mm
* Infill: 20–30%
* Supports: Only where required
